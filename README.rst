Single-page Django Admin using django-reset-framework, React,
and other front-end frameworks via plugins.

Documentation
-------------

The full documentation is at https://django-rest-admin.readthedocs.org.

Quickstart
----------

Install django-rest-admin::

    pip install django-rest-admin

Then use it in a project::

    import rest_admin
